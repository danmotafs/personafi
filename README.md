# PersonaFi

PersonaFi is a Web3 infrastructure built on Solana that empowers informal merchants with:

- Digital payments
- Token-based loyalty
- AI-powered identity

## MVP

- Create Persona
- Support Merchant
- Simple Dashboard

## Stack

- Solana + Anchor
- Next.js
- Node.js

## Status

🚧 MVP in progress (Hackathon)
