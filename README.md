# PersonaFi

PersonaFi is a financial infrastructure on Solana that empowers informal merchants in Brazil with digital identity, payments, and loyalty systems.

---

## 🚨 Problem

Millions of informal merchants in Brazil operate outside the digital economy.

They lack:
- Access to digital payments
- Customer retention tools
- Online presence

---

## 💡 Solution

PersonaFi turns real-world merchants into onchain businesses.

Each merchant gets:
- AI-powered digital identity
- Solana-based payments
- Loyalty tokens (SPL)

---

## ⚙️ How it works

1. Merchant creates a persona
2. A token is minted on Solana
3. Customers can support/pay the merchant
4. Engagement is tracked onchain

---

## 🧱 Tech Stack

- Solana + Anchor
- SPL Token + Metaplex
- Next.js + TypeScript
- Node.js backend
- Supabase/PostgreSQL
- AI for persona generation
